for crypto page api ----  https://api.coingecko.com/api/v3/coins/markets?vs_currency=${currency}&ids=${coinSearch}&order=${sort}&per_page=10&page=${page}

for pagination total number of pages ----- https://api.coingecko.com/api/v3/coins/list

for search result useing coin name api ----- https://api.coingecko.com/api/v3/search?query=${query}

for trending coins api ---- https://api.coingecko.com/api/v3/search/trending
